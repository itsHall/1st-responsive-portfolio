MORGANTOWN RUNNING

website redesign

Morgantown Running: Website Redesign is a proposed redesign of a local business that specializes in retail footwear. It was a great challenge in copywriting, web design, and usability.

This project required a lot of planning and consideration. I broke this 6 week project down into 7 parts: Site Inventory, Card Sorting, IA Diagram, Wireframes, Mockups, Style Tiles, Coding (HTML & CSS), and User Testing. A Site Inventory is basically mapping the information on the current site, recording each page and it's content. I used an activity called Card Sorting to ask potential site users how they would arrange the information in the site. I recorded their amount of exercise and internet use. I arranged each of these "tests" into a better presentation I could use to determine how to create the layout of information on the site.

An IA (Information Architecture) Diagram is a plan for the navigation and location of information on the site. Changes can occur from this stage but this is a strong foundation for mockups and coding that follow later. Once I have an idea of how information will be placed on the site's pages, I can test some layouts by creating Wireframes that show the bones of the site's design.

After I have wireframes, the next stage is Mockups in Adobe Illustrator. I created 5 pages to be used as templates (different page designs to be used across the site.) Mocking the site up allows me to determine the appropriate colors, fonts, and styles to use in the project. To arrange this for myself and the client, I created a Style Tile.

All of this preparation led to the meat of the project: the Coding. I coded the site in semantic HTML using a 16 column grid with the CSS 960 Grid System. Finally, to reveal any bugs or mistakes, I show the coded site to fresh eyes for an outside opinion with User Testing. The users were asked to complete various tasks and answer questions about the site. I recorded the screen as well as the user's face in order to observe how they navigated and interacted. I organized the results for myself and the client and fixed the discovered issues.
